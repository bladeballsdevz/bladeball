### GUI Doesn't Appear After Injection

- Confirm the executor is fully updated
- Try pressing the default toggle key (`RightShift`)
- Re-inject after fully closing and reopening the game

```bash

```

- Ensure `auto_parry.enabled` is `true` in config

### All Swords Not Switching

- Verify `all_swords.enabled` is `true`
- Confirm the sword priority list includes currently owned swords
- Re-log into the game after config changes

### Script Crashes on Android

- Clear the loader app cache
- Reinstall the APK with the latest version
- Disable battery optimization for the loader app
